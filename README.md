Project Overview:

• This project focuses on Exploratory Data Analysis (EDA) of the Zomato restaurant dataset to uncover insights about restaurant   trends, customer ratings, pricing, and service availability across different locations.

• The objective is to perform data cleaning, analysis, and visualization to support data-driven decision making for the food      and restaurant industry.

Objectives:

• Understand restaurant distribution across countries and cities
• Analyze restaurant ratings and customer voting patterns

• Study cost trends and affordability of restaurants

• Explore the impact of online delivery and table booking on ratings

• Identify popular cuisines and high-rated restaurant categories

Tools & Technologies Used:

• Programming Language: Python

• Libraries:

   Pandas – Data manipulation
   
   NumPy – Numerical operations
   
   Matplotlib & Seaborn – Data visualization
   
• Environment: Google Colab

Project Outcome:

This project demonstrates the ability to:

  • Perform structured EDA on real-world datasets
  
  • Extract meaningful business insights
  
  • Create clear and informative visualizations
  
  • Apply data analysis skills in a practical scenario
